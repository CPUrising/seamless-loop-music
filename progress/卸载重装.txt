1. 它的默认地址
您可以直接把下面这段路径粘贴进文件夹路径栏（或者按 Win + R 输入）：

text
%LOCALAPPDATA%\uv
具体路径通常是：C:\Users\您的用户名\AppData\Local\uv\cache。

2. 怎么精准找到它？
如果您想让 uv 自己老老实实交代位置，可以在终端输入这个命令：

cmd下
uv cache dir

这个命令会清空 uv 的所有临时工具缓存。

cmd下
uv cache clean

1. 简单的预下载命令
在大人的命令行（CMD 或 PowerShell）里输入：

powershell
uvx pymusiclooper --version
这条命令会让 uv 自动去下载并配置好所有运行环境。一旦这条命令运行成功并显示了版本号，莱芙的程序就能在下次启动时自动扫描到它，再也不会弹出那个讨人厌的确认框了！(๑>◡<๑)

2. 如果指定下载位置
如果您想让 uv 把环境手动下到指定的目录D:\program，可以用这两步：

PowerShell 版：

powershell
$env:UV_CACHE_DIR="D:\program\uv_cache"
uvx pymusiclooper --version

CMD 版：

cmd
set UV_CACHE_DIR=D:\program\uv_cache
uvx pymusiclooper --version
