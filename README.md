# Seamless Loop Music Player (无缝循环音乐播放器)

## 📖 项目简介
本软件是一款专为游戏背景音乐 (BGM) 或循环素材设计的音频处理工具。它支持通过精确的采样点设置，确保 `.wav`, `.mp3`, `.ogg` 等格式音频在循环播放时达到完美的“无缝”效果。现已迁移至 .NET Framework 4.8，具备更强的 Windows 系统兼容性。

## 🚀 使用方法
1. **导入资源**：点击“导入音乐文件夹”，选择包含音频文件的目录。
2. **选择曲目**：在右侧列表中双击或使用“上/下一首”切换音频。
3. **调整循环点**：
    - 在输入框中直接输入采样数或秒数。
    - 使用下方的按钮（如 `-1s`, `+500` 等）进行微调。
4. **验证效果**：点击“确认应用并试听”，程序将应用设置并自动跳转至循环结束点前 3 秒以便校验。
5. **重播功能**：点击“重新播放”可随时瞬间回到音频 0 秒位置。
6. **自动保存**：所有循环点设置将自动保存至本程序目录下的配置文件中。
