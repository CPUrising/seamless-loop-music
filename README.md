## English Description

A professional audio playback and fine-tuning tool designed specifically for game music (BGM) or loop materials. It supports precision sample point settings to ensure perfect seamless transitions.

### âœ¨ Key Features
- **Maximum Compatibility**: Built on **.NET Framework 4.8**. Native support for Windows 10/11 without the need for additional runtime installations.
- **Precision Loop Control**: Adjust loop points via **Samples** or **Seconds** for frame-perfect accuracy.
- **Confirmation Mechanism**: Apply loop settings through the "Apply & Preview" button to prevent audio stutters during real-time adjustments.
- **Smart Preview**: One-click jump to 3 seconds before the loop end point to quickly verify the transition quality.
- **Instant Replay**: Instantly jump back to the 0s position and restart playback for overall listening comparisons.
- **Multi-format Support**: Seamless loops for `.wav`, `.mp3`, and `.ogg` files.
- **Bilingual UI**: One-click toggle between Chinese and English.
